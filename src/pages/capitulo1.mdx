# Bem-vindo ao modelo de livro acessível digital

Este é um modelo de livro acessível digital que utiliza a tecnologia de
leitura acessível para tornar o conteúdo mais acessível para todos os
usuários.

## Por que MDX?

MDX nos permite escrever conteúdo em Markdown enquanto mantemos a capacidade
de usar componentes React quando necessário. Isso torna o conteúdo mais
fácil de escrever e manter.

### Vantagens

- **Simplicidade**: Escreva em Markdown puro
- **Flexibilidade**: Use componentes React quando precisar
- **Consistência**: Layout padrão aplicado automaticamente
- **Manutenibilidade**: Separação clara entre conteúdo e código
